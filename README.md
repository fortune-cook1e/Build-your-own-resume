# Resume Generator

Build your resume online

## Features

1. Create as many resumes as you like

## Get Started

```bash
docker-compose up -d

# generate Prisma client
pnpm run prisma:generate

# run it when prisma schema updates
pnpm run prisma:migrate:dev

# development
pnpm run dev

# build
pnpm run build

# For development
# board port: 5173
# server port: 3000
# web port: 8080
# chrome compose: 3003

```
